# Travel Planner - Frontend (React + Tailwind CDN)

This is a frontend-only React project scaffold for **Travel Planner**.
Features included:
- Login Page
- Signup Page
- Dashboard
- Plan New Trip
- My Trips Page
- Budget Planner Page
- Settings Page

How to run:
1. Copy the project folder into your local machine or extract the zip.
2. Run `npm install`.
3. Run `npm start` and open http://localhost:3000 in VS Code Live Preview or browser.

Notes:
- Tailwind is loaded via CDN in `public/index.html` using `https://cdn.tailwindcss.com`.
